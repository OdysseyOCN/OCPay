# ZWPlaceHolder
一行代码实现UITextView的placeHolder
##如何使用
- cocoapods导入(搜索不到请更新本地仓库)
```
pod   'ZWPlaceHolder'
```
- 直接将文件拖入工程中，引入头文件即可
```
#import "UITextView+ZWPlaceHolder.h"
```
